编辑器 - Sublime Text

Posted on 2019-08-30 Edited on 2023-05-26 In iTools

Sublime Text 是一款跨平台文本编辑器，可以通过 packages 进行扩展。社区维护了大量的免费扩展包可供使用。

Write the code, Change the world.

Package Control

安装（推荐）

  1. 打开 Tools 菜单
  2. 选择 Install Package Control...

使用

  1. 打开命令面板
    command + shift + p
  2. 输入 install package，按 enter
  3. 输入你想安装的 package 名称，按 enter

推荐的 package

Common

  • Alignment: 轻松对齐多个选择和多行选择
  • All Autocomplete: 扩展 sublime 的自动完成功能，会在当前窗口所有打开的文件中检索匹配的内容
  • BracketHighlighter: 用于 Sublime Text 括号和标签高亮
  • GitGutter: 可以在 gutter 中看到 git diff
  • ini: 对 INI 和 REG 文件进行语法高亮显示
  • SublimeLinter: 代码提示框架

Python

  • Pylinter: pylint 插件，结合 Pylint 使用

FE

  • Babel: ES6 JavaScript 的语法定义

  • JsFormat: Javascript 格式化工具，需要在用户偏好设置中设置

Theme

  • Theme - Soda: 安装完成后，需要在用户偏好设置中设置

Shortcuts

  • command + enter: 在当前行下面新增一行
  • command + shift + enter: 在当前行上面新增一行
  • command + shift + L: 将光标分别添加到选中的行尾
  • command + shift + ← 或者 command + shift + →: 快速选中一行的其余部分
  • control + G: 快速跳转到某一行
  • control + shift + K: 快速删除当前行
  • F5: 排序
