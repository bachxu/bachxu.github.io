代码绘图工具 PlantUML

Posted on 2023-09-07 Edited on 2023-09-08 In iTools

众所周知 ProcessOn 是一款好用的在线作图工具，其实 PlantUML 也是一个允许你通过代码进行快速绘图的组件，如果你深度偏好代码绘图，那么 PlantUML 是值得推荐的。

本文主要介绍如何在本地搭建代码可视化绘图环境。

准备条件

  • Visual Studio Code

  • macOS 基于 homebrew 快速安装

本地 PlantUML Server 渲染器安装

PlantUML Server 是一个 web 应用，用于即时生成 UML 图表。我们可以通过 Docker 在本地进行部署。

  1. 通过 docker pull plantuml-server 镜像

  2. 可以选择 jetty 或者 tomcat 创建并运行 plantuml 容器

  3. 访问站点 http://localhost:8080
