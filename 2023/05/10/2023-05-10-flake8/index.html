Flake8 简单使用

Posted on 2023-05-10 Edited on 2023-06-29 In iTools

Flake8 一款 Python 工具，它将 pycodestyle、pyflakes、mccabe 和第三方插件集成在一起，以检查一些 python 代码的风格和质量。

安装

  1. 激活虚拟环境

  2. 安装 flake8

在 PyCharm 内设置扩展工具

PyCharm -> Perferences -> Tools -> External Tools, Select +

配置信息

  • Program: $PyInterpreterDirectory$/python
  • Arguments: -m flake8 --show-source --statistics $ProjectFileDir$
  • Working directory: $ProjectFileDir$
  • Output Filter: left empty

使用方法

选择 Tools -> External Tools -> Flake8

Flake 配置文件

在项目根目录，创建一个命名为 .flake8 的文件，添加自定义规则
