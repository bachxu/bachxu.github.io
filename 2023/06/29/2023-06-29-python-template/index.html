Python 的 string - Template 使用指南

Posted on 2023-06-29 In iTechnology

Template 是 Python 中一种字符串格式化工具：模版字符串。这是一种更加简单且功能较弱的机制，但在某些情况下这可能正是我们所寻找的。

简单使用

Template 属于 string 中的一个类，所以使用的时候需要先从 string 标准库中导入

Template 需要格式化的字符串有个特殊标识符 **$**，可以是 $xxx，也可以是 **${xxx}**，其中 xxx 是满足 python 命名规则的字符串，不能以数字开头，也不能为关键字等。

实现原理

Template 的实现方式是首先通过 Template 初始化一个字符串，这些字符串中包含了一个个 key。通过调用 substitute 或 safe_substitute，将 key 值与方法中传递过来的参数匹配上，从而实现指定的位置导入字符串。这种方式的好处是不用像 "Hello, %s" % "Bob" 之类的方式，各个参数的顺序必须固定，只要 key 是正确的，值就能正确替换。

substitute 是一个严肃的方法，如果有 key 没有输入，那就一定会报错，可以通过来发现问题

重写 Template

我们还可以通过继承重写的方式，对 Template 进行一些个性化的修改。只需修改 delimiter 字段就可以将 $ 字符修改为其他字符，如 **#**，不过新的标识符需要符合正则表达式的规范。通过修改 idpattern 可以修改 key 的命名规则，比如说规定第一个字符开头必须是 a，这对规范命名可以起约束作用，当然这也是通过正则表达实现的。
