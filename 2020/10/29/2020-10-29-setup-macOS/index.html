Homebrew

Homebrew 安装苹果（或 Linux 系统）没有你所需要的东西。

下载最新版本的 Python

背景

众所周知， Python 2 自 2020 年 1 月 1 日起停止维护。苹果从 macOS 12.3 开始不再与 python2.7 进行捆绑，但值得注意的是 python3 也没有预安装在 macOS 上。我们可以通过安装 Xcode，其中包括了 python3。

下载与安装

在 Python.org 下载适配 macOS 的最新版本。

Poetry

介绍

Poetry 是 Python 用来管理依赖性和打包的工具。它允许你声明你项目所依赖的库，它将为你管理（安装/更新）它们。

系统要求

Poetry 需要 python3.5 以上版本。它是跨平台的，目标是让它在 Windows、Linux 和 macOS 上都能运行使用。

安装

Poetry 提供了一个定制的安装程序，它可以将 poetry 与你系统的其它部分隔离安装。这是推荐的安装 poetry 方法。
