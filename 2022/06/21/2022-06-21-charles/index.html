人生苦短, 需要 Charles

Posted on 2022-06-21 Edited on 2023-06-29 In iTools

Charles 是一款集 HTTP proxy/HTTP monitor/Reverse Proxy 于一体，使调试能够快速、可靠和先进，节省时间的工具。

关键特点

  • SSL 代理 – 查看纯文本的SSL请求和响应
  • 带宽节流，模拟较慢的互联网连接，包括延时场景
  • AJAX 调试 – 以树状或文本形式查看 XML 和 JSON 请求和响应
  • AMF – 以树状形式查看 Flash Remoting/Flex Remoting 消息的内容
  • 重复请求以测试后端变化
  • 编辑请求以测试不同的输入
  • 断点 - 拦截和编辑请求或响应
  • 使用 W3C 验证器验证记录的 HTML、CSS 和 RSS/atom 响应

安装

背景

  • macOS 12.4

从官网下载安装包并安装。

使用

SSL 代理

如果你只捕获 HTTP 请求，只需要打开默认配置的 Charles。如果你捕获 HTTPS 请求，可以参考以下步骤。

安装 Charles 根证书

  1. 选择 Charles 菜单栏路径 Help -> SSL Proxying -> Install Charles Root Certificate
  2. 点击以上菜单后，会自动打开 Keychain Access 应用
  3. 如果该证书不被信任，那么名称旁边的图标是红色的。双击 Charles 的密钥串，在使用证书时选择 Always Trust

SSL 代理设置

  1. 选择 Charles 菜单栏路径 Proxy -> SSL Proxying Settings
  2. 打开 Enable SSL Proxying 开关，添加 *:443 到 Include 栏
  3. 从 Charles 菜单路路径 Proxy -> Proxy Settings，打开 Enable transparent HTTP proxying 开关

PC/Web 浏览器

选择 Charles 菜单栏路径 Proxy -> macOS Proxy, 然后就可以捕获从 PC/Web 浏览器的请求。

Mobile 设备

只要打开 External Proxy Servers 就可以捕获从 Mobile 设备的请求。

  1. 取消选择 Charles 菜单栏路径 Proxy -> macOS Proxy
  2. 在状态栏点击 Charles 应用图标, 选择 External Proxy Servers

要设置 Mobile 设备代理，需要注意设备和电脑所连接的是同一个 Wi-Fi。

  1. 在 iPhone 端打开 Settings 应用
  2. 在 WLAN 选择连接的网络
  3. 配置代理

捕获 HTTPS 请求

如果你想捕获手机上的 HTTPS 请求，需要选择 Charles 菜单栏路径 Help -> SSL Proxying -> Install Charles Root Certificate on a Mobile Device or Remote Browser，然后会弹出一个对话框，上面会描述移动设备上的具体配置步骤。

FAQ

当有 VPN 直连时，Charles 不能捕获请求

  1. 下载并安装 Chrome 的扩展应用 Proxy SwitchyOmega
  2. 打开 SwitchyOmega 扩展并且新建一个配置
  3. 配置协议，服务和端口，并且应用配置
  4. 打开 Charles, 只选择 External Proxy Servers，不要选择 macOS Proxy
  5. 当你要捕获请求时，在 SwitchyOmega 切换到 charles

如果你不捕获请求了，不要忘记把 SwitchyOmega 切换回 System Proxy
