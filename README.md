# 单人即每

这是一个[博客](https://bachxu.github.io/)，主要记录生活的感想以及技术的积累。文章目前没有体系结构，完全依托于时间节点（现阶段给自己定的时间节点是每天一个节点，主要培养自己去表达想法这个习惯）。

## 框架

博客构建比较简单，主要基于 [Github](https://github.com) 中的 Pages 服务和 [Jekyll](https://jekyllrb.com/)。

## 反馈

欢迎大家到 [Issues](https://github.com/bachxu/bachxu.github.io/issues) 中发表自己的感想和见解。
